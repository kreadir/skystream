## Changelogs - v2.2.7

### ✨ New Features

- 🔎 **Plugin-wise Search**  
  Perform searches within specific plugins for more accurate and relevant results.

- 💡 **Search Suggestions**  
  Get intelligent suggestions while typing to discover content faster.

- 🔁 **Multi-Provider Support in a Single Plugin**  
  Plugins can now handle multiple content providers, improving availability and reliability.

- 🌐 **Domain Switching Support**  
  Easily switch between domains if supported by the plugin.

- 🌍 **Expanded TMDB Language Support (40+ Languages)**  
  Enhanced Explore filters with broader language options.

- 🪵 **In-App Logs Screen (For Plugin Developers)**  
  View detailed logs directly within the app to debug and improve plugins more efficiently.

- 📺 **tvOS Support**  
  SkyStream is now available on Apple TV for a seamless big-screen experience.
